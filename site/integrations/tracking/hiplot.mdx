Polyaxon comes with a [Hiplot](https://facebookresearch.github.io/hiplot/index.html) integration to analyze multiple experiments.

## Tracking

You can track hyperparameter and metrics of your experiments using [Polyaxon's tracking module](/docs/tracking/tracking/module).

## In notebooks

You can use the module [MultiRunPlot](/docs/tracking/visualizations/programmatic/#hiplot) which requires hiplot to create interactive parallel coordinates inside a notebook or jupyter lab.

## Example

```python
from polyaxon.client import RunClient

client = RunClient()
exp = client.get_runs_as_hiplot(query="metrics.loss:<0.1")
exp.display()
```
