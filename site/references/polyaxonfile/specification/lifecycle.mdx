## V1Statuses

```python
polyaxon._schemas.lifecycle.V1Statuses()
```


Status is the information that represents the current state of a run.

By examining a run status and/or the history of its statuses,
you can learn what stage the run is at, and what stages are left.

The supported statuses by Polyaxon.

- __Enum__:
  - __CREATED__: "created"
  - __ON_SCHEDULE__: "on_schedule"
  - __RESUMING__: "resuming"
  - __WARNING__: "warning"
  - __UNSCHEDULABLE__: "unschedulable"
  - __COMPILED__: "compiled"
  - __QUEUED__: "queued"
  - __SCHEDULED__: "scheduled"
  - __STARTING__: "starting"
  - __RUNNING__: "running"
  - __SUCCEEDED__: "succeeded"
  - __FAILED__: "failed"
  - __UPSTREAM_FAILED__: "upstream_failed"
  - __STOPPING__: "stopping"
  - __STOPPED__: "stopped"
  - __SKIPPED__: "skipped"
  - __RETRYING__: "retrying"
  - __UNKNOWN__: "unknown"
