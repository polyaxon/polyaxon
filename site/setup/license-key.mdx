# Enterprise License Key

All core Polyaxon features and APIs are available in Polyaxon CE without any limits.

When running Polyaxon self-hosted, you use the same deployment infrastructure as Polyaxon Cloud. There are no scalability limitations between the different versions.

Some additional Enterprise features require a license key:

- [Project-level RBAC Roles](/docs/administration/rbac)
- [Data Retention Policies](/docs/administration/data-retention)
- [Audit Logs](/docs/administration/audit-logs)
- UI Customization
- Organization Creators
- [Org Management API and SCIM](/docs/administration/scim-and-org-api)
- [Instance Management API](/docs/administration/scim-and-org-api)

See [pricing page](/pricing-self-host) for more details on Polyaxon Enterprise.

## Activating a License Key

After purchasing a license key, you can activate it by adding the `organizationKey` field to your Polyaxon deployment chart:

```yaml
organizationKey: <your-license-key>
```
