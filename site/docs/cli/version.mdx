### version

```bash
Usage: polyaxon version [OPTIONS]

  Print the current version of the cli and platform.

Options:
  --check  Check compatibility versions.
  --help   Show this message and exit.
```

---

### upgrade

```bash
Usage: polyaxon upgrade [OPTIONS]

  Install/Upgrade polyaxon cli.

Options:
  --help  Show this message and exit.
```
