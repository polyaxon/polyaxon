## Overview

The comparison view is a powerful tool for listing, filtering, sorting, rearranging, and comparing runs.


## Features

You can control the columns to show or hide on the table, you can control their order, and the width of each column:

![comparison-columns](/images/dashboard/comparison/columns.png)


You can filter and sort by many columns:

![comparison-sort](/images/dashboard/comparison/sort.png)

For runs with metrics, you can switch to a visualization mode to compare metrics charts, curve charts:

![comparison-scatter](/images/dashboard/comparison/charts-scatter.png)

![comparison-roc-curves](/images/dashboard/comparison/charts-roc-curves.png)

![comparison-lines](/images/dashboard/comparison/charts-lines.png)

![comparison-many](/images/dashboard/comparison/charts-many.png)

![comparison-multi](/images/dashboard/comparison/charts-multi.png)

![comparison-compare](/images/dashboard/comparison/charts-compare.png)

![comparison-bars](/images/dashboard/comparison/charts-bars.png)

The comparison features are also useful to create charts for hyperparameter tuning or a selection of runs to create hyperparameter values and model metrics charts

![comparison-new-charts](/images/dashboard/comparison/charts-new.png)

And compare runs on Tensorboard

![comparison-tensorboard](/images/dashboard/comparison/tensorboard.png)

![comparison-tensorboard-compare](/images/dashboard/comparison/tensorboard-compare.png)
