## Overview

Polyaxon dashboard provides several visualizations and charts based on events logged by the [tracking API](/docs/tracking/tracking/).

## Media rendering

Polyaxon can render several media types, you can log several versions of the same media with step index:

 * images
 * text
 * html
 * audio
 * video
 * dataframe
